## Фильм на вечер
Эта консольная ```ruby``` программа возвращает случайный фильм из википедии или локального хранилища по выбранному режиссеру.

### Использование

Установите ```Bundler```
```
gem install bundler
```
В папке программы установите гемы:
```
bundle install
```
Запустите программу:
```
bundle exec ruby main.rb
```
